# Trinity ROI Calculator — Customer Version

## Project Overview
Customer-facing ROI calculator for Trinity Automation. Customers fill in their info, see their ROI, and submit for a proposal. Lead data sent to jonmadden@selwaytool.com via Web3Forms.

**Stack:** Vanilla HTML/CSS/JS

## Brand
Same as sales rep version: charcoal (#2D2D2D) + gold (#D4A843) + Trinity branding.

## Key Differences from Sales Rep Version
- Customer-friendly language ("What's Your Machine Really Worth?")
- Lead capture form (name, company, email, phone)
- Web3Forms integration (key: b00ba4eb-0735-408c-bc0f-192d4c094285)
- Sends to: jonmadden@selwaytool.com
- PDF proposals use all-black text

## Never Do
- Remove the Web3Forms lead capture
- Change the recipient email without approval
- Remove the customer-friendly language
- Reference Selway Machine Tool in the customer-facing UI
